Viewing Party

If you want to watch things online with your friends. This project can help you to create a series of functions based on things you've watched, favorited, and want to watch and your friends haven't watched, or things that your friends have watched and you haven't watched. Can even make recommendations!

Swap Meet

This program is useful to organize a swap meet for you and your friends to swap your stuff! For this event, you want each person to register online as a vendor. Also, they should list an inventory list of things. Vendors can swap items between different inventories

Go to Github

Task List API

With this web API to organize our tasks, we'll be able to create, read, update, and delete tasks! Some of the features this API can make are: sorting tasks, marking them as complete, get feedback about our task list through Slack, organize tasks with goals and more!

Go to Github

Adagrams

This program is a game in which a player is given a random set of letters and must make an anagram using those letters. Each submitted word will score points.
